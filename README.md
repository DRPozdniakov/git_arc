# AI Infrastructure Landscape 2026

**Interactive map of 340+ AI/ML tools across 28 categories.**

[**View Live**](https://drpozdniakov.github.io/ai-infrastructure-map/)

## What It Is

A force-directed graph visualization of the entire AI/ML infrastructure ecosystem — model providers, frameworks, vector databases, GPU clouds, observability tools, and more.

- **340+ tools** with clickable links to official websites
- **28 categories** including LLM Orchestration, Agent Frameworks, Fine-Tuning, Model Serving, MCP Ecosystem
- **200+ official icons** via LobeHub and Simple Icons CDNs
- **Dual filter system** — filter by type (LLM, Training, Infra, etc.) and company size (Large/Medium/Small/Startup)
- **Search** — find any tool instantly
- **Bubble sizing** — reflects company scale ($1B+ to Startup/OSS)

## How to Use

1. Open [the live page](https://drpozdniakov.github.io/ai-infrastructure-map/)
2. **Scroll** to zoom in/out
3. **Drag** nodes to rearrange
4. **Click** any tool to visit its website
5. Use **filters** on the left to narrow by category or company size
6. Use the **search bar** to find specific tools

## Tech Stack

- D3.js v7 — force-directed graph simulation
- Single self-contained HTML file (no build step)
- GitHub Pages hosting
