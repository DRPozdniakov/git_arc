# AI Infrastructure Landscape 2026

**Interactive map of 440+ AI/ML tools across 28 categories.**

[**View AI Map**](https://drpozdniakov.github.io/git_arc/)

## What It Is

A visualization of the entire AI/ML infrastructure ecosystem — model providers, frameworks, vector databases, GPU clouds, robotics, observability tools, and more.

- **440+ tools** with clickable links to official websites
- **28 categories** including LLM Orchestration, Agent Frameworks, Fine-Tuning, Model Serving, MCP Ecosystem, Robotics AI
- **200+ official icons** via LobeHub and Simple Icons CDNs
- **Two views** — Blueprint (force-directed graph) and Cards (categorized grid)
- **Dual filter system** — filter by type and company size
- **Search** — find any tool instantly
- **Bubble sizing** — reflects company scale ($1B+ to Startup/OSS)

## How to Use

1. Open [the live page](https://drpozdniakov.github.io/git_arc/)
2. Switch between **Blueprint** and **Cards** views
3. **Scroll** to zoom in/out, **drag** nodes to rearrange
4. **Click** any tool to visit its website
5. Use **filters** to narrow by category or company size
6. Use the **search bar** to find specific tools

## Tech Stack

- D3.js v7 — force-directed graph simulation
- Self-contained HTML files (no build step)
- GitHub Pages hosting
